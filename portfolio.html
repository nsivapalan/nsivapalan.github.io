Twitch Streams

This is the Twitch Streams project, built as part of FCC's Intermediate Front-end Development projects. From a list of specified Twitch channels, it displays which channels are currently live (streaming), and which are offline and inactive, with buttons to display them.

Used: HTML, CSS, Bootstrap, JavaScript, jQuery, and the Twitch API.

Wikipedia Viewer

This is the Wikpedia Viewer project, built as part of FCC's Intermediate Front-end Development projects. It is a search tool that displays the top results from Wikipedia based on your search term. It also lets you search for a random article.

Used: HTML, CSS, Bootstrap, JavaScript, jQuery, and the MediaWiki action API.

Local Weather App

This is the Local Weather App project, built as part of FCC's Intermediate Front-end Development projects. It displays the weather (temperature and conditions) in both Fahrenheit and Celsius for you at your current location using the Open Weather API and IP API for quicker access to your location.

Used: HTML, CSS, Bootstrap, JavaScript, jQuery, Open Weather API, and the IP API.

Random Quote Machine

This is the Random Quote Machine project, built as part of FCC's Intermediate Front-end Development projects. It generates a quote for your daily dose of inspiration! It also lets you tweet the quote and author.

Used: HTML, CSS, Bootstrap, JavaScript, jQuery, and the Forismatic API.
